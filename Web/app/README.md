# Веб-интерфейс ресторана

React/Vite-приложение для публичного сайта и веб-админки проекта.

## Запуск

```bat
npm install
npm run dev
```

## Роуты

- `/` — главная страница меню
- `/menu` — меню
- `/wines` — винная карта
- `/book` — бронь
- `/info` — информация о ресторане
- `#/admin` — админка

## Локальный backend

По умолчанию Vite проксирует запросы на `http://127.0.0.1:8000` через `/api` и `/photos`.

Если backend запущен на другом адресе, поправьте `vite.config.ts`.
