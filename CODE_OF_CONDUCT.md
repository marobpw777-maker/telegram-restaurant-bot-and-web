# Code of Conduct

Этот проект придерживается простого правила: уважительное, корректное и рабочее взаимодействие.

## Ожидаемое поведение

- не переходить на личности;
- не публиковать секреты и приватные данные;
- писать понятные issue и pull request;
- указывать шаги воспроизведения для багов;
- держать тон деловым и предметным.

## Недопустимо

- оскорбления;
- публичная публикация чувствительных данных;
- намеренный саботаж рабочих сценариев;
- обсуждение уязвимостей без осторожности, если они затрагивают реальные данные.
